<h1>BallDontLie.io API</h1>

<p>Ever curious about your favorite basketball player or team stats? Well this is the code for you!</p>
<p>This program fetches data from the "BallDontLie.io" and "FREE NBA" API to beautifully display basketball statistics from 1979 till present day. It does this by using a combination of HTML, CSS, JavaScript and React.js. The program also uses router-dom to help navigate from one component to another displaying the different pages of statistical data.</p>
